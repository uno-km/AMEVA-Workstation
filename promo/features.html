Core Capabilities

기존 협업 도구들을 완전히 대체하는 혁신적인 문서 워크스테이션

카드를 클릭하여 각 기능의 상세한 메커니즘과 비즈니스 활용 방안을 확장 확인하세요.

AI Copilot & Diff Acceptor

자연어 명령 편집 및 Git 스타일 충돌 분석 UI 탑재.

사용자가 본문을 드래그하고 "수정해줘" 라고 전송하면, AI가 수정 전/후를 분석하여 Diff 카드로 띄웁니다. 사용자는 클릭 한 번으로 변경 제안을 본문에 즉시 융합(Accept)하거나 기각(Reject)할 수 있습니다.

4대 언어 가상 샌드박스

문서 내에서 JavaScript, Python, HTML, SQL 즉시 컴파일 실행.

별도의 개발 서버 환경 구축 필요 없이 에디터 내부의 Jupyter Block을 통해 런타임 결과와 오류 로그를 즉시 수집하고 테이블 양식으로 보기 좋게 렌더링합니다.

Y.js 분산 실시간 협업

로컬 P2P 및 퍼블릭 릴레이 클라우드 듀얼 모드 CRDT 싱크.

외부 서버 연동 없이 동일 인트라넷 망 내에서 독자적인 웹소켓 서버를 1초 만에 띄워 동료들을 초대할 수 있으며, 공용 릴레이 서버로 전 세계 팀원들과 지연률 0%로 문서와 캐럿을 공유합니다.

리치 미디어 임베더

고화질 비디오, 오디오 스트리밍 블록 완벽 지원.

용량이 큰 오디오 파일이나 사내 시연 비디오 가이드라인을 문서 본문 내에 깔끔하게 임베딩하고 커스텀 플레이어 위젯을 통해 편리하게 청취 및 인라인 재생을 할 수 있습니다.

통합 플러그인 마켓플레이스

업무 템플릿과 다양한 확장 도구 원클릭 조회 및 로드.

앱 내부의 마켓플레이스 메뉴를 통해 다양한 서식 템플릿, 커스텀 확장 블록, 생산성 플러그인 생태계를 원클릭으로 주입하여 에디터의 기능을 필요에 따라 무한히 늘려나갈 수 있습니다.

HWP(아래한글) & 원노트 기능 확장

한글(HWP) 및 원노트 스타일 생산성 모듈 완벽 지원.

국내 문서 환경에 꼭 맞는 아래한글(HWP) 스타일 서식 뷰와 원노트(OneNote) 같은 강력하고 유연한 페이지 구조 확장 모듈을 클릭 한 번으로 설치 및 이식할 수 있어 한 차원 높은 편의성을 선사합니다.

하이브리드 다중 문서 모드

덮어쓰기, 이어서 열기, 탭별 열기를 조율하는 스마트 멀티뷰.

작업의 성격에 따라 파일을 별도의 탭으로 나누어 보거나(Tab Mode), 하나의 에디터 아래에 연속으로 이어 붙여(Append Mode) 하나의 긴 문서처럼 스크롤하며 편집할 수 있습니다. 각 문서의 경계는 사이드바에 앵커 링크로 표출되어 클릭 한 번으로 이동할 수 있습니다.

지능형 AI MCP 도구 확장

사용자 정의 MCP 서버를 플러그인하여 AI 에이전트 자동화 실행.

HTTP 및 stdio 기반의 임의의 MCP 서버를 마우스 클릭 몇 번으로 실시간 연동할 수 있습니다. 이를 통해 로컬/원격 AI 에이전트가 로컬 파일 제어, 외부 API 호출, 데이터베이스 직접 쿼리 등의 실질적인 액션을 직접 실행하게 만들어 문서 작성 및 개발 생산성을 비약적으로 극대화합니다.
